### Magnolia Rental Booking System
Django web application for managing short-term rental property bookings and availability calendar.

## About
This project was developed to solve real business needs for my family-run short-term rental business in Macedonia. It handles the complete booking workflow from availability tracking to customer management, replacing manual processes and third-party platform dependencies.

## Features
Booking Management: Track and manage reservations with calendar integration
Availability Calendar: Real-time room availability tracking and updates
Customer Data Management: Store and organize guest information
Automated Workflows: Streamline check-in/check-out and booking confirmations
Multi-language Support: Interface designed for international clientele

## Tech Stack
